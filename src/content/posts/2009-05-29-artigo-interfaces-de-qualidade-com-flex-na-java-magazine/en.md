My article about Java and Flex, **_Quality Interfaces with Adobe Flex_**, was published in Java Magazine.

It was written for readers who knew some Java and were starting with Flex. The article compared ActionScript 3.0 with Java 5, demonstrated ways to consume data, and explained how code created for the web with Flash Player could also run on the desktop without changing its functional logic.

At the time, I was also preparing a follow-up about corporate Flex applications integrated with Spring Framework, EJB, and JBoss.
