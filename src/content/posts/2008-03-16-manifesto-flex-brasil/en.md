This manifesto collected my main contributions to the Brazilian Adobe Flex community through March 2008.

## Articles and projects

- **[Setting Up a Flex 2 Environment and Hello World](http://e-digows.blogspot.com/2007/03/configurando-um-ambiente-para.html):** an introduction to setting up the development environment for beginners.
- **[A Simple RSS Reader](http://e-digows.blogspot.com/2007/03/flex-leitor-de-rss.html):** an RSS reader built in a few lines with `HTTPService`.
- **[Adobe Flex: Playing with Effects](http://e-digows.blogspot.com/2007/04/adobe-flex-e-efeitos.html):** a demonstration of effects applied to Flex components.
- **[Exchanging Objects Between Flex and PHP](http://e-digows.blogspot.com/2007/11/trocando-objetos-flex-com-php-via.html):** using VO, DTO, and TO objects with `RemoteObject` and a PHP back end.
- **[Adobe Flex Developer Census](/2007/11/16/how-was-adobe-flex-doing-in-brazil-in-2007/):** an application created to map Flex developers in Brazil.
- **[Working with CSS in Flex Builder 3](/2007/12/29/working-with-css-in-adobe-flex-builder-3/):** creating CSS styles for Flex components.
- **[Architecture with Adobe Flex, PHP, and Zend Framework](/2008/01/03/architecture-with-adobe-flex-php-and-zend-framework/):** a layered organization and practices for Flex applications backed by PHP.
- **FlexBrasil Forum:** an alternative to the Flex-Brasil and FlexDev mailing lists for preserving discussions and helping new developers. The original forum is offline.
- **Integrating Flex, BlazeDS, Spring Framework, and Hibernate:** a two-part tutorial about `RemoteObject` and Java frameworks.

As part of the manifesto chain, the next invitation went to [Eberton Consolim](http://blog.flexdev.com.br/manifesto-flex-brasil/).
